============================================================
  税理士事務所 AIエージェント管理システム - Windows起動手順
============================================================

【ステップ1】Pythonのインストール（初回のみ）
  1. Chromeで https://www.python.org/downloads/ を開く
  2. 「Download Python 3.12.x」ボタンをクリックしてダウンロード
  3. ダウンロードしたファイルをダブルクリックで実行
  4. ★重要★「Add Python to PATH」のチェックボックスに必ずチェックを入れる
  5. 「Install Now」をクリック
  6. インストール完了後、PCを再起動（推奨）

【ステップ2】プロジェクトフォルダを展開（初回のみ）
  1. 「Windows転送用_AIシステム.zip」を右クリック
  2. 「すべて展開」を選択
  3. 展開先を「Cドライブ直下」または「書類フォルダ」に設定
     例: C:\zeirishi_ai\
  4. 展開ボタンをクリック

【ステップ3】APIキーの設定（初回のみ）
  1. 展開したフォルダ内の「backend」フォルダを開く
  2. 「env_template」ファイルを右クリック →「名前の変更」で「.env」に変更
     ※ 先頭がドット（.）になることを確認
  3. .env ファイルを右クリック →「メモ帳で開く」
  4. 「ここにAPIキーを貼り付けてください」の部分を削除し、
     Anthropic APIキー（sk-ant-api03-...）を貼り付ける
  5. Ctrl+S で保存して閉じる

【ステップ4】セットアップ実行（初回のみ）
  1. 展開したフォルダ内の「setup_windows.bat」をダブルクリック
  2. 「このアプリがデバイスに変更を加えることを許可しますか？」→「はい」
  3. 黒い画面が開き、インストールが始まる（数分待つ）
  4. 「セットアップ完了！」と表示されたらEnterキーを押して閉じる

【ステップ5】サーバー起動（毎日の起動手順）
  1. 「start_servers.bat」をダブルクリック
  2. 黒い画面が2つ開く（バックエンド・フロントエンド）
  3. Chromeで http://localhost:5173 が自動的に開く
  4. ダッシュボードが表示されれば起動成功！

【ステップ6】終了時
  1. 「stop_servers.bat」をダブルクリックしてサーバーを停止
  2. または黒い画面を×で閉じる

============================================================
【よくあるエラー】

エラー: 「'python'は内部コマンドまたは外部コマンド、操作可能なプログラムまたは
       バッチファイルとして認識されていません」
→ Pythonの「Add Python to PATH」にチェックを入れてから再インストール

エラー: 「APIキーが設定されていません」（ブラウザ上）
→ backend\.env ファイルのAPIキーを確認する

エラー: ページが開かない / 繋がらない
→ stop_servers.bat を実行後、start_servers.bat を再実行
